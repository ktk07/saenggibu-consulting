# 진로 정체성 생성 기준

## 1. 좋은 진로 정체성이란?

좋은 진로 정체성은 단순한 직업명이 아니라 학생이 관심 있는 대상과 해결하고 싶은 문제를 포함해야 한다.

좋은 진로명은 다음 형식을 따른다.

수식어 + 직업

예시:
- 고령사회 법률 사각지대를 해결하는 노인전문 변호사
- 재생에너지 확대에 따른 전력망 안정성을 연구하는 전기공학자
- 데이터 기반으로 도시 교통 불평등을 개선하는 산업공학자

## 2. 좋은 진로 정체성의 조건

1. 직업명이 포함된다.
2. 관심 분야가 드러난다.
3. 해결하고 싶은 문제가 드러난다.
4. 학생의 가치관이나 기여 방향이 드러난다.
5. 이후 교과 탐구로 연결 가능하다.
6. 관련 학과와 연결 가능하다.

## 3. 약한 진로 정체성

- 변호사
- 의사
- 개발자
- 공학자
- 경영인
- 연구원

이런 표현은 너무 넓기 때문에 좋은 진로 정체성이 아니다.

## 4. 진로명 생성 공식

진로명 =
관심 대상 + 해결하고 싶은 문제 + 사용하는 도구 + 기여 방식 + 직업

## 5. 진로 후보 생성 원칙

1. 학생의 입력을 무시하지 않는다.
2. 너무 생소하거나 비현실적인 직업은 피한다.
3. 직업명보다 문제의식을 먼저 드러나게 한다.
4. 같은 직업이라도 서로 다른 분야를 제안한다.
5. 학생이 선택하고 조정할 수 있도록 여러 후보를 제시한다.
